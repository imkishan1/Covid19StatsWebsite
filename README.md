# Covid19StatsWebsite
CovidStats provides the most recent information on covid19 in India. It gets real-time data from the covid19bharat.org API (earlier covid19India.org). The website displays the current case count in the country, as well as the number of active cases and overall number of patients recovered since Covid19.
